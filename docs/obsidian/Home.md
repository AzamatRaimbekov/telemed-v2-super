# MedCore KG — Телемедицинская SaaS Платформа

> Комплексная телемедицинская платформа для клиник Кыргызстана. Многопользовательская система с электронными медицинскими картами, видеоконсультациями, голосовым ассистентом, мониторингом пациентов и управлением зданием.
