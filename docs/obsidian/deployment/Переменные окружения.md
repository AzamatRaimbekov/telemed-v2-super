# Переменные окружения

Файл: `.env` (на основе `.env.example`)

## База данных (PostgreSQL)

| Переменная | Описание |
|-----------|----------|
| `POSTGRES_HOST` | Хост БД |
| `POSTGRES_PORT` | Порт (5432) |
| `POSTGRES_DB` | Имя базы данных |
| `POSTGRES_USER` | Пользователь |
| `POSTGRES_PASSWORD` | Пароль |

## Redis

| Переменная | Описание |
|-----------|----------|
| `REDIS_HOST` | Хост Redis |
| `REDIS_PORT` | Порт (6379) |

## MinIO (S3)

| Переменная | Описание |
|-----------|----------|
| `MINIO_ENDPOINT` | URL MinIO |
| `MINIO_ACCESS_KEY` | Ключ доступа |
| `MINIO_SECRET_KEY` | Секретный ключ |

## JWT Аутентификация

| Переменная | Описание |
|-----------|----------|
| `JWT_SECRET_KEY` | Секрет для подписи JWT |
| `ACCESS_TOKEN_EXPIRE_MINUTES` | Время жизни токена |
| `ALGORITHM` | Алгоритм (HS256) |

## SMTP (Email)

| Переменная | Описание |
|-----------|----------|
| `SMTP_HOST` | SMTP сервер |
| `SMTP_USER` | Email отправителя |
| `SMTP_PASSWORD` | Пароль |

## AI/Voice

| Переменная | Описание |
|-----------|----------|
| `GEMINI_API_KEY` | Google Gemini API key |
| `DEEPSEEK_API_KEY` | DeepSeek API key |
| `WHISPER_MODEL_SIZE` | Размер модели Whisper |

## Приложение

| Переменная | Описание |
|-----------|----------|
| `APP_ENV` | Окружение (dev/prod) |
| `DEBUG` | Режим отладки |
| `CORS_ORIGINS` | Разрешённые домены |
