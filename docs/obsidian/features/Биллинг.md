# Биллинг

Финансовый модуль: счета, платежи, финансовая аналитика.

## Модели

- **Invoice** — счёт (пациент, сумма, статус, дата)
- **InvoiceItem** — строка счёта (услуга, количество, цена)
- **Payment** — платёж (сумма, метод, дата)

## Процесс

```
Услуга оказана → InvoiceItem создан → Invoice сформирован →
    Пациент видит в портале → Payment зафиксирован
```
