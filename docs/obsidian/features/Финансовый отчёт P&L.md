# Финансовый отчёт P&L

Финансовый отчёт по месяцам: выручка, выставленные счета, количество счетов, процент собираемости. Дневная выручка.

## Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/reports/financial/pl` | P&L отчёт по месяцам |
| GET | `/api/v1/reports/financial/daily` | Дневная выручка за последние N дней |

## Метрики (помесячно)

- **revenue** — полученные платежи (сом)
- **billed** — выставленные счета (сом)
- **invoices_count** — количество счетов
- **collection_rate** — процент собираемости (revenue / billed)

## Параметры

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| months | int | 6 | Количество месяцев (макс. 24) |
| days | int | 30 | Количество дней для daily (макс. 90) |
