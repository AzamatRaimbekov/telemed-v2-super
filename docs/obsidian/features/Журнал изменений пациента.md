# Журнал изменений пациента

## Описание

Детальный журнал изменений данных пациента с diff-представлением. Отслеживает создание, изменение и удаление записей: карта пациента, визиты, рецепты, результаты анализов. Встраиваемый компонент для страницы пациента.

## Модели БД

### `patient_changelogs`
| Поле | Тип | Описание |
|------|-----|----------|
| patient_id | UUID FK | Пациент |
| changed_by_id | UUID FK | Кто изменил |
| changed_by_name | String(200) | Имя изменившего |
| action | String(50) | create / update / delete |
| entity_type | String(50) | patient / visit / prescription / lab_result |
| entity_id | UUID | ID изменённой сущности |
| changes | JSON | Diff: {"field": {"old": "...", "new": "..."}} |
| summary | Text | Человекочитаемое описание |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/patients/{id}/changelog` | История изменений | Staff |
| POST | `/patients/{id}/changelog` | Записать изменение | Staff |
