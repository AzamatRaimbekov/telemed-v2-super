# Предиктивная аналитика

Прогнозирование загрузки палат, потока пациентов и расхода лекарств на основе исторических данных.

## Прогнозы

### 1. Загрузка палат (7 дней)
- 14-дневная скользящая средняя по BedAssignment
- Коэффициент дня недели (середина недели выше)
- Доверительный интервал ±15%
- Визуализация: AreaChart

### 2. Поток пациентов (7 дней)
- 14-дневная скользящая средняя по Appointment
- Выходные × 0.3
- Визуализация: BarChart

### 3. Расход лекарств (30 дней)
- Оценка дневного расхода из Inventory
- Алерт когда запас закончится
- Уровни: critical (≤3 дня), warning (≤7 дней), info (≤30 дней)
- Визуализация: карточки с прогресс-барами

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/predictions/beds?days=7` | Прогноз палат |
| GET | `/api/v1/predictions/admissions?days=7` | Прогноз приёмов |
| GET | `/api/v1/predictions/medications?days=30` | Прогноз лекарств |
| GET | `/api/v1/predictions/dashboard` | Все прогнозы одним запросом |
