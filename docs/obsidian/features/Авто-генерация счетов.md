# Авто-генерация счетов после визита (Auto-Billing)

## Описание

Автоматическое создание счёта (Invoice) после завершения визита пациента. Врач или система указывает оказанные услуги, и счёт формируется с уникальным номером, позициями и итоговой суммой. Также доступны шаблоны цен на типовые услуги.

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/v1/auto-billing/generate` | Создать счёт после визита |
| GET | `/api/v1/auto-billing/templates` | Получить шаблоны цен на услуги |
| POST | `/api/v1/auto-billing/templates` | Создать/обновить шаблон цены |

## Тело запроса — генерация счёта

```json
{
  "patient_id": "uuid",
  "visit_id": "uuid (опционально)",
  "services": [
    {"name": "Консультация терапевта", "price": 500.0, "quantity": 1},
    {"name": "ЭКГ", "price": 300.0, "quantity": 1}
  ]
}
```

Если `services` не указаны, создаётся счёт по умолчанию: "Консультация врача" — 500 сом.

## Сервис

**Файл:** `backend/app/services/auto_billing.py`

- `AutoBillingService.generate_invoice_for_visit()` — создаёт Invoice + InvoiceItem записи
- `AutoBillingService.get_service_templates()` — возвращает типовые шаблоны цен
- Автоматическая генерация номера счёта: `INV-YYYYMMDD-NNNN`
