# Страховые компании (Insurance Companies)

## Описание

Управление страховыми компаниями, полисами пациентов и страховыми заявками. Проверка валидности полиса, отслеживание лимитов покрытия, одобрение/отклонение заявок на возмещение.

## Модели БД

### `insurance_companies`
| Поле | Тип | Описание |
|------|-----|----------|
| name | String(200) | Название компании |
| code | String(20) | Уникальный код |
| contact_person | String(200) | Контактное лицо |
| phone | String(50) | Телефон |
| email | String(200) | Email |
| address | Text | Адрес |
| contract_number | String(100) | Номер договора |
| contract_valid_until | Date | Срок действия договора |
| discount_percent | Float | Процент скидки |
| is_active | Boolean | Активна |
| covered_services | JSON | Покрываемые услуги |

### `insurance_policies`
| Поле | Тип | Описание |
|------|-----|----------|
| patient_id | UUID FK | Пациент |
| company_id | UUID FK | Страховая компания |
| policy_number | String(50) | Номер полиса (индексирован) |
| valid_from | Date | Начало действия |
| valid_until | Date | Окончание действия |
| coverage_type | String(50) | Тип покрытия (standard/premium/vip) |
| max_amount | Float | Максимальная сумма |
| used_amount | Float | Использованная сумма |
| is_active | Boolean | Активен |

### `insurance_claims`
| Поле | Тип | Описание |
|------|-----|----------|
| policy_id | UUID FK | Полис |
| invoice_id | UUID | Счёт (опционально) |
| claim_amount | Float | Сумма заявки |
| approved_amount | Float | Одобренная сумма |
| status | String(20) | pending / approved / rejected / paid |
| rejection_reason | Text | Причина отказа |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/insurance/companies` | Список страховых компаний | Staff |
| POST | `/insurance/companies` | Создать компанию | Staff |
| PUT | `/insurance/companies/{id}` | Обновить компанию | Staff |
| DELETE | `/insurance/companies/{id}` | Удалить компанию (soft) | Staff |
| GET | `/insurance/policies` | Список полисов | Staff |
| POST | `/insurance/policies` | Создать полис | Staff |
| GET | `/insurance/patient/{patient_id}` | Активные полисы пациента | Staff |
| GET | `/insurance/verify/{policy_number}` | Проверка валидности полиса | Staff |
| GET | `/insurance/claims` | Список заявок | Staff |
| POST | `/insurance/claims` | Подать заявку | Staff |
| PATCH | `/insurance/claims/{id}/approve` | Одобрить заявку | Staff |
| PATCH | `/insurance/claims/{id}/reject` | Отклонить заявку | Staff |
