# Мультиязычность (i18n)

> Поддержка кыргызского и русского языков в интерфейсе MedCore KG.

---

## Технологии

| Пакет | Назначение |
|-------|-----------|
| `i18next` | Ядро интернационализации |
| `react-i18next` | React-биндинги (хук `useTranslation`) |
| `i18next-browser-languagedetector` | Автоопределение языка браузера |

## Поддерживаемые языки

| Код | Язык | Статус |
|-----|------|--------|
| `ru` | Русский | Основной (fallback) |
| `ky` | Кыргызский | Полная поддержка |

## Файловая структура

```
frontend/src/
  lib/i18n.ts                          — Инициализация i18next
  locales/
    ru.json                            — Русские переводы
    ky.json                            — Кыргызские переводы
  components/shared/
    language-switcher.tsx               — Компонент переключения языка
```

## Как использовать

### В компонентах

```tsx
import { useTranslation } from "react-i18next";

function MyComponent() {
  const { t } = useTranslation();
  return <h1>{t("dashboard.title")}</h1>;
}
```

### Переключение языка

Компонент `<LanguageSwitcher />` расположен в верхней панели (header) рядом с уведомлениями. Выбранный язык сохраняется в `localStorage` под ключом `medcore_language`.

## Структура переводов

Переводы организованы по разделам:

- `common` — общие слова (сохранить, отмена, удалить и т.д.)
- `auth` — авторизация
- `nav` — навигация (sidebar)
- `patients` — раздел пациентов
- `dashboard` — главная панель
- `roles` — роли пользователей
- `portal` — портал пациента

## Добавление нового языка

1. Создать файл `frontend/src/locales/{code}.json` по образцу `ru.json`
2. Добавить импорт и ресурс в `frontend/src/lib/i18n.ts`
3. Добавить кнопку в `frontend/src/components/shared/language-switcher.tsx`
