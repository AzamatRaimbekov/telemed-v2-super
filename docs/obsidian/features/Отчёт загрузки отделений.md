# Отчёт загрузки отделений

Отчёт по загруженности отделений: количество коек, занятость, процент загрузки.

## Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/reports/departments/load` | Метрики загрузки по каждому отделению |
| GET | `/api/v1/reports/departments/summary` | Общая сводка по всем отделениям |

## Метрики

- **total_beds** — общее количество коек
- **occupied_beds** — занятые койки
- **available_beds** — свободные койки
- **occupancy_rate** — процент загрузки (%)
