# Виджеты дашборда

Кастомизация главной панели — пользователь выбирает какие виджеты показывать.

## Доступные виджеты
Пациенты, Приёмы, Палаты, Доход, Лаборатория, Аптека, Задачи, Инфекции, Очередь, Прогнозы

Состояние сохраняется в localStorage.

**Файл:** `frontend/src/components/shared/dashboard-widgets.tsx`
