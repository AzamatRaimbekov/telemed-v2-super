# Реабилитация

Комплексная система реабилитации — инсульт, упражнения с MediaPipe, цели восстановления.

## Модули

### Инсульт-реабилитация
- **StrokeAssessment** — оценка по шкалам NIHSS, mRS
- **RehabGoal** — цели реабилитации
- **RehabProgress** — отслеживание прогресса

### Упражнения (MediaPipe)
- **Exercise** — каталог 15+ упражнений для реабилитации
- **ExerciseSession** — сессия тренировки
- **ExerciseRep** — каждое повторение (углы суставов, правильность формы)
- Камера отслеживает позу в реальном времени
- Подсчёт повторений, оценка качества

### Цели восстановления
- **RecoveryGoal** — цели по доменам (моторика, речь, когнитив)
- **RecoveryDomainWeight** — веса доменов для общего индекса
- Визуализация прогресса
