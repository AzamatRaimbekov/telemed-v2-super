# Система ожидания (Wait Time Estimation)

> Расчёт примерного времени ожидания пациента в очереди. Показывает количество людей впереди, расчётное время и публичное табло для клиники.

---

## Обзор

Система использует существующую модель `QueueEntry` для расчёта:
- Количество людей впереди в очереди
- Среднее время обслуживания (15 мин/пациент)
- Расчётное время ожидания в минутах

---

## Архитектура

### Бэкенд

- **Маршруты:** `backend/app/api/v1/routes/wait_time.py`
- Использует модель `QueueEntry` и `QueueStatus`
- Новых моделей не требуется

---

## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/wait-time/estimate/{queue_id}` | Оценка ожидания для конкретного пациента |
| GET | `/api/v1/wait-time/display/{clinic_id}` | Публичное табло времени ожидания |

### Ответ estimate

```json
{
  "queue_number": 5,
  "people_ahead": 3,
  "estimated_minutes": 45,
  "estimated_time": "~45 мин"
}
```

### Ответ display

```json
{
  "total_waiting": 8,
  "avg_wait_minutes": 120,
  "updated_at": "2026-04-23T10:30:00Z"
}
```
