# Корпоративные договоры (Corporate Contracts)

## Описание

Управление договорами с организациями. Учёт компаний-клиентов, контрактов с лимитами (сотрудники, сумма), скидки, реестр сотрудников, верификация принадлежности к организации, контроль использования бюджета.

## Модели БД

### `corporate_contracts`
| Поле | Тип | Описание |
|------|-----|----------|
| company_name | String(200) | Название компании |
| company_inn | String(20) | ИНН |
| contact_person | String(200) | Контактное лицо |
| contact_phone | String(50) | Телефон |
| contact_email | String(200) | Email |
| contract_number | String(50) | Номер договора (уникальный) |
| valid_from | Date | Начало действия |
| valid_until | Date | Окончание действия |
| max_employees | Integer | Макс. кол-во сотрудников |
| max_amount | Float | Лимит суммы |
| used_amount | Float | Использованная сумма |
| discount_percent | Float | Скидка (%) |
| covered_services | JSON | Покрываемые услуги |
| is_active | Boolean | Активен ли договор |
| notes | Text | Примечания |

### `corporate_employees`
| Поле | Тип | Описание |
|------|-----|----------|
| contract_id | UUID FK | Договор |
| patient_id | UUID FK | Привязанный пациент |
| employee_name | String(200) | ФИО сотрудника |
| employee_id_number | String(50) | Табельный номер |
| position | String(100) | Должность |
| is_active | Boolean | Активен |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/corporate/contracts` | Список договоров (фильтр: active_only, search) | Staff |
| POST | `/corporate/contracts` | Создать договор | Staff |
| PUT | `/corporate/contracts/{id}` | Обновить договор | Staff |
| DELETE | `/corporate/contracts/{id}` | Удалить договор (soft) | Staff |
| GET | `/corporate/contracts/{id}/employees` | Сотрудники по договору | Staff |
| POST | `/corporate/contracts/{id}/employees` | Добавить сотрудника | Staff |
| GET | `/corporate/contracts/{id}/usage` | Отчёт использования (бюджет, сотрудники, дни) | Staff |
| POST | `/corporate/verify-employee` | Верифицировать сотрудника по ФИО/номеру | Staff |
