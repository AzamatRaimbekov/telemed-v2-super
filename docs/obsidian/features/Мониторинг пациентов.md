# Мониторинг пациентов

Система реального времени для отслеживания жизненных показателей пациентов.

## Типы датчиков

| Тип | Описание |
|-----|----------|
| HEART_RATE | Пульс |
| SPO2 | Сатурация кислорода |
| BODY_TEMPERATURE | Температура тела |
| BLOOD_PRESSURE | Артериальное давление |
| FALL_DETECTOR | Детектор падений |
| RESPIRATORY_RATE | Частота дыхания |
| ECG | Электрокардиограмма |
| GLUCOSE | Глюкоза крови |
| WEIGHT | Вес |
| ACTIVITY | Активность |

## Типы алертов

| Алерт | Критичность |
|-------|------------|
| FALL_DETECTED | Critical |
| HIGH_HEART_RATE | Warning/Critical |
| LOW_SPO2 | Critical |
| HIGH_TEMPERATURE | Warning |
| LOW_TEMPERATURE | Warning |
| HIGH_BP | Warning |
| LOW_BP | Warning |
| ABNORMAL_ECG | Critical |

## Протокол

1. Датчики отправляют показания через API
2. Сервер проверяет пороги (thresholds)
3. При превышении — создаётся MonitoringAlert
4. WebSocket транслирует обновления на фронтенд
5. Медсестра получает уведомление
6. NurseCall — кнопка вызова медсестры у койки
