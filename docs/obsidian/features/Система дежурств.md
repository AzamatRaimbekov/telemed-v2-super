# Система дежурств

Управление графиком дежурств врачей: назначение, просмотр по дням и неделям, фильтрация по отделению.

## Возможности

- Назначение дежурства врачу на конкретную дату
- Типы дежурств: дневное, ночное, суточное, на вызове
- Время начала и окончания смены
- Привязка к отделению
- Просмотр: сегодня / завтра / неделя
- Недельный вид с группировкой по дням
- Фильтрация по врачу, типу, отделению, диапазону дат
- Мягкое удаление (soft delete)

## Типы дежурств (DutyType)

| Тип | Описание |
|-----|----------|
| `day` | Дневное |
| `night` | Ночное |
| `full` | Суточное (24ч) |
| `on_call` | На вызове |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/duty-schedule/` | Назначить дежурство |
| GET | `/api/v1/duty-schedule/` | Список (фильтры: date_from, date_to, doctor_id, duty_type, department) |
| GET | `/api/v1/duty-schedule/today` | Кто дежурит сегодня |
| GET | `/api/v1/duty-schedule/tomorrow` | Кто дежурит завтра |
| GET | `/api/v1/duty-schedule/week` | Недельный вид |
| PATCH | `/api/v1/duty-schedule/{id}` | Обновить |
| DELETE | `/api/v1/duty-schedule/{id}` | Удалить |
| GET | `/api/v1/duty-schedule/doctor/{doctor_id}` | Дежурства конкретного врача |
