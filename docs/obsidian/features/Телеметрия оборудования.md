# Телеметрия оборудования (Equipment Telemetry)

## Описание

Учёт медицинского оборудования с телеметрией: ИВЛ, мониторы пациента, инфузоматы, дефибрилляторы, КТ/МРТ, лабораторные анализаторы. Приём показаний датчиков, алерты, контроль ТО, дашборд состояния оборудования.

## Модели БД

### `medical_equipment`
| Поле | Тип | Описание |
|------|-----|----------|
| name | String(200) | Наименование |
| serial_number | String(100) | Серийный номер (уникальный) |
| equipment_type | Enum | ventilator/patient_monitor/infusion_pump/defibrillator/ecg_machine/xray/ultrasound/ct_scanner/mri/lab_analyzer/autoclave/other |
| manufacturer | String(200) | Производитель |
| model | String(200) | Модель |
| location | String(200) | Расположение |
| room_id | UUID | Палата (опционально) |
| status | Enum | operational/maintenance/repair/offline/decommissioned |
| last_maintenance | DateTime | Последнее ТО |
| next_maintenance | DateTime | Следующее ТО |
| hours_used | Integer | Часы наработки |
| is_connected | Boolean | Подключено (онлайн) |

### `equipment_readings`
| Поле | Тип | Описание |
|------|-----|----------|
| equipment_id | UUID FK | Оборудование |
| reading_type | String(50) | Тип показания (temperature, pressure, flow_rate, battery, error_code) |
| value | Float | Числовое значение |
| unit | String(20) | Единица измерения |
| string_value | String(200) | Строковое значение |
| is_alert | Boolean | Является алертом |
| metadata_json | JSON | Доп. данные |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/equipment/` | Список оборудования (фильтр: type, status, search) | Staff |
| POST | `/equipment/` | Зарегистрировать оборудование | Staff |
| PUT | `/equipment/{id}` | Обновить оборудование | Staff |
| PATCH | `/equipment/{id}/status` | Изменить статус | Staff |
| POST | `/equipment/{id}/reading` | Отправить показание телеметрии | Staff |
| GET | `/equipment/{id}/readings` | Показания оборудования (limit, reading_type) | Staff |
| GET | `/equipment/alerts` | Активные алерты (за N часов) | Staff |
| GET | `/equipment/maintenance-due` | Оборудование, требующее ТО (7 дней) | Staff |
| GET | `/equipment/dashboard` | Обзорная статистика | Staff |
