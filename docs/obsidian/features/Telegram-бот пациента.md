# Telegram-бот пациента

> Telegram-бот для пациентов: запись на приём, проверка анализов, информация о клинике.

---

## Обзор

Бот позволяет пациентам взаимодействовать с клиникой через Telegram — записываться на приём, проверять результаты анализов, получать информацию о клинике и напоминания о лекарствах.

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/запись` | Запись к врачу |
| `/анализы` | Проверка результатов анализов |
| `/приёмы` | Предстоящие приёмы |
| `/лекарства` | Напоминания о лекарствах |
| `/клиника` | Информация о клинике (адрес, телефон, часы) |
| `/помощь` | Список всех команд |
| `/отмена` | Отмена записи |

## Архитектура

### Backend

- **Сервис:** `backend/app/services/patient_telegram_bot.py` — класс `PatientTelegramBot`
- **Роут:** `backend/app/api/v1/routes/patient_telegram.py` — webhook endpoint
- **Endpoint:** `POST /api/v1/patient-telegram/webhook`

### Привязка аккаунта

Пациент отправляет свой ИНН для привязки Telegram-аккаунта к профилю в системе.

## Настройка

1. Создать бота через [@BotFather](https://t.me/BotFather)
2. Добавить `TELEGRAM_BOT_TOKEN` в `.env`
3. Настроить webhook: `https://api.medcore.kg/api/v1/patient-telegram/webhook`
