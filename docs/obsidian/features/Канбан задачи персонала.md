# Канбан задачи персонала

## Описание

Система управления задачами персонала в формате Kanban-доски. Четыре колонки: Сделать, В работе, На проверке, Готово. Поддержка drag & drop, приоритетов и назначения ответственных.

## Модели БД

### `staff_tasks`
| Поле | Тип | Описание |
|------|-----|----------|
| title | String(300) | Название задачи |
| description | Text | Описание |
| status | Enum | todo / in_progress / review / done |
| priority | Enum | low / medium / high / urgent |
| assigned_to_id | UUID FK | Назначенный сотрудник |
| created_by_id | UUID FK | Автор задачи |
| patient_id | UUID FK | Связанный пациент (опционально) |
| due_date | Date | Срок выполнения |
| sort_order | Integer | Порядок сортировки в колонке |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/tasks/board` | Канбан-доска (сгруппировано по статусу) | Staff |
| GET | `/tasks/` | Список задач с фильтрами | Staff |
| POST | `/tasks/` | Создать задачу | Staff |
| PATCH | `/tasks/{id}` | Обновить задачу | Staff |
| PATCH | `/tasks/{id}/move` | Переместить в колонку | Staff |
| DELETE | `/tasks/{id}` | Мягкое удаление | Staff |
