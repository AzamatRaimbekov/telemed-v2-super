# DICOM интеграция

Регистрация, хранение и описание медицинских снимков (DICOM-исследований): рентген, КТ, МРТ, УЗИ и др.

## Возможности

- Регистрация DICOM-исследования (study) с привязкой к пациенту
- Назначение направляющего врача и рентгенолога
- Фильтрация по модальности, статусу, дате, пациенту
- Добавление заключения рентгенолога (report)
- Автосмена статуса при добавлении заключения
- Просмотр всех снимков пациента
- Список доступных модальностей

## Модальности (Modality)

| Код | Описание |
|-----|----------|
| `CR` | Рентген |
| `CT` | КТ (компьютерная томография) |
| `MR` | МРТ (магнитно-резонансная томография) |
| `US` | УЗИ (ультразвук) |
| `XA` | Ангиография |
| `MG` | Маммография |
| `DX` | Цифровой рентген |
| `OTHER` | Прочее |

## Статусы

| Статус | Описание |
|--------|----------|
| `uploaded` | Загружено |
| `processing` | Обрабатывается |
| `ready` | Готово (есть заключение) |
| `archived` | Архивировано |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/dicom/studies` | Зарегистрировать исследование |
| GET | `/api/v1/dicom/studies` | Список (фильтры: patient_id, modality, status, date_from, date_to) |
| GET | `/api/v1/dicom/studies/{id}` | Детали с заключением |
| PATCH | `/api/v1/dicom/studies/{id}/report` | Добавить/обновить заключение |
| GET | `/api/v1/dicom/patient/{patient_id}` | Все снимки пациента |
| GET | `/api/v1/dicom/modalities` | Справочник модальностей |
