# Авто-назначение палаты (Auto Room Assignment)

## Описание

Автоматический подбор и назначение свободной койки пациенту при госпитализации. Сервис учитывает отделение, доступность коек и статус. Поддерживает как предложение (suggest) без бронирования, так и полное авто-назначение с созданием BedAssignment и обновлением статуса койки.

## Сервис

### `AutoAssignmentService`
- `find_best_bed(clinic_id, gender, department_id, isolation_required)` — поиск лучшей свободной койки
- `auto_assign_patient(patient_id, clinic_id, gender, department_id)` — авто-назначение с созданием записи

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| POST | `/auto-assign/bed` | Авто-назначить пациенту койку | Staff |
| POST | `/auto-assign/suggest` | Предложить лучшую койку (без назначения) | Staff |

### Тело запроса `/auto-assign/bed`
```json
{
  "patient_id": "uuid",
  "department_id": "uuid (опционально)",
  "gender": "male/female (опционально)"
}
```

### Ответ
```json
{
  "bed_id": "uuid",
  "room_id": "uuid",
  "bed_number": "А-101",
  "room_name": null,
  "auto_assigned": true
}
```
