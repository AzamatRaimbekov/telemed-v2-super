# Напоминания о задачах персонала

> Celery-задача проверяет просроченные и приближающиеся задачи каждый час. Уведомления отправляются через WebSocket.

---

## Логика

Каждый час Celery beat запускает задачу `check_task_reminders`:

1. **Просроченные задачи** — `due_date < сегодня`, статус TODO или IN_PROGRESS
2. **Завтрашние задачи** — `due_date == завтра`, статус TODO или IN_PROGRESS

Для каждой найденной задачи отправляется WebSocket-уведомление назначенному сотруднику.

## Типы уведомлений

| Тип | WebSocket event | Сообщение |
|-----|----------------|-----------|
| Просрочена | `task_overdue` | «Просрочена задача: {title}» |
| Завтра дедлайн | `task_reminder` | «Завтра дедлайн: {title}» |

## Celery Beat расписание

```python
"check-task-reminders": {
    "task": "check_task_reminders",
    "schedule": 3600.0,  # каждый час
}
```

## WebSocket payload

```json
{
  "type": "task_overdue",
  "task_id": "uuid",
  "title": "Название задачи",
  "due_date": "2026-04-23",
  "message": "Просрочена задача: Название задачи"
}
```
