# Дневник медсестры (UI)

Фронтенд для ежедневных записей медсестринского наблюдения.

## Страница `/nurse-diary`
- Поиск по пациенту
- Записи с указанием смены (дневная, вечерняя, ночная) и состояния (удовлетворительное, средней тяжести, тяжёлое, критическое)
- Создание записи: витальные (температура, АД, пульс, ЧДД, SpO2), сознание, жалобы, процедуры, лекарства, диета, заметки

- `frontend/src/features/nurse-diary/api.ts`
- `frontend/src/routes/_authenticated/nurse-diary.tsx`
