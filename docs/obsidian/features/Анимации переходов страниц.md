# Анимации переходов страниц

> Плавные анимации при переходе между страницами (fade + slide).

---

## Обзор

Компонент `PageTransition` оборачивает содержимое страницы и добавляет анимацию появления/исчезновения с помощью `framer-motion`.

## Параметры анимации

| Свойство | Значение |
|----------|---------|
| Initial | opacity: 0, y: 8px |
| Animate | opacity: 1, y: 0 |
| Exit | opacity: 0, y: -8px |
| Duration | 0.2s |
| Easing | easeOut |

## Использование

```tsx
import { PageTransition } from "@/components/shared/page-transition";

export function MyPage() {
  return (
    <PageTransition>
      <h1>Содержимое страницы</h1>
    </PageTransition>
  );
}
```
