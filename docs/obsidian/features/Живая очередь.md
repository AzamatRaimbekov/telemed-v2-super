# Живая очередь приёма

Реалтайм очередь пациентов с табло для холла.

## Компоненты

### Управление (для регистратуры)
- Добавление пациента в очередь
- Вызов следующего
- Начать/Завершить/Пропустить приём
- Автообновление каждые 5 сек

### Лобби-дисплей (публичный)
- Полноэкранный тёмный фон для ТВ
- Крупные номера очереди, читаемые с 3-5 метров
- Группировка по кабинетам
- Часы реального времени
- Автообновление каждые 3 сек
- URL: `/lobby?clinic_id=...`

## Статусы очереди

| Статус | Описание |
|--------|----------|
| `waiting` | Ожидает вызова |
| `called` | Вызван |
| `in_progress` | На приёме |
| `completed` | Завершён |
| `skipped` | Пропущен |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/queue/add` | Добавить в очередь |
| POST | `/api/v1/queue/call-next` | Вызвать следующего |
| POST | `/api/v1/queue/{id}/start` | Начать приём |
| POST | `/api/v1/queue/{id}/complete` | Завершить |
| POST | `/api/v1/queue/{id}/skip` | Пропустить |
| GET | `/api/v1/queue/today` | Очередь на сегодня |
| GET | `/api/v1/queue/lobby/{clinic_id}` | Лобби-дисплей (публичный) |
