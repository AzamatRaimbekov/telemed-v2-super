# Фармация

Управление аптекой, лекарствами, рецептами и складом.

## Модули

### Каталог лекарств
- Название, форма выпуска, дозировка
- Производитель, страна
- АТС классификация

### Рецепты
- Назначение врачом
- Дозировка, частота, длительность
- Статус: активный/завершён/отменён

### Складской учёт
- Остатки по лекарствам
- Срок годности
- Минимальный запас → автозаказ

### Поставщики
- Реестр поставщиков
- Контактные данные
- История поставок

### Заказы поставщикам
- Формирование заказа
- Статус: черновик/отправлен/доставлен
- Приёмка товара
