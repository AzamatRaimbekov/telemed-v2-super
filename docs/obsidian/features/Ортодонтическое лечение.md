# Ортодонтическое лечение (Orthodontic Tracking)

> Ведение ортодонтических случаев: брекеты, элайнеры, прогресс визитов, оплата, фото прогресса.

---

## Архитектура

### Backend
- **Модель:** `backend/app/models/ortho_treatment.py`
  - `OrthoTreatment` — лечение (тип, даты, визиты, оплата, элайнеры, фото)
  - `OrthoType` — enum (braces_metal, braces_ceramic, braces_sapphire, braces_lingual, aligners)
- **Маршруты:** `backend/app/api/v1/routes/ortho.py`
  - `POST /ortho/` — начать лечение
  - `GET /ortho/patient/{patient_id}` — лечения пациента
  - `GET /ortho/{id}` — детали
  - `PATCH /ortho/{id}` — обновить
  - `POST /ortho/{id}/visit` — записать визит (инкремент visits_completed)
  - `POST /ortho/{id}/photo` — добавить фото прогресса
  - `GET /ortho/{id}/progress` — сводка прогресса

## Прогресс-сводка

Endpoint `GET /ortho/{id}/progress` возвращает:
- `progress_percent` — % завершения (визиты/план)
- `months_elapsed` / `months_remaining` — время лечения
- `aligner_progress` — "5/20" (текущий/всего) для элайнеров
- `total_cost` / `paid_amount` / `remaining_payment` — финансы
- `payment_percent` — % оплаты
- `photos_count` — количество фото прогресса

## Типы ортодонтического лечения

| Тип | Описание |
|-----|----------|
| braces_metal | Металлические брекеты |
| braces_ceramic | Керамические брекеты |
| braces_sapphire | Сапфировые брекеты |
| braces_lingual | Лингвальные брекеты |
| aligners | Элайнеры (каппы) |
