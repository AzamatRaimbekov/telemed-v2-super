# Складской учёт расходников (Consumables Inventory)

## Описание

Учёт медицинских расходных материалов: шприцы, перчатки, бинты, иглы, катетеры, маски и др. Отслеживание остатков, контроль минимального запаса, учёт расхода по отделениям, контроль сроков годности.

## Модели БД

### `consumable_items`
| Поле | Тип | Описание |
|------|-----|----------|
| name | String(200) | Наименование |
| sku | String(50) | Артикул (уникальный) |
| category | Enum | syringes/gloves/bandages/needles/catheters/masks/gowns/disinfectant/sutures/other |
| unit | String(20) | Единица измерения (шт, уп, л, мл) |
| quantity | Integer | Текущий остаток |
| min_quantity | Integer | Минимальный запас |
| unit_price | Float | Цена за единицу |
| supplier | String(200) | Поставщик |
| expiry_date | Date | Срок годности |
| location | String(100) | Место хранения |
| notes | Text | Примечания |

### `consumable_usages`
| Поле | Тип | Описание |
|------|-----|----------|
| item_id | UUID FK | Расходник |
| used_by_id | UUID FK | Кто использовал |
| patient_id | UUID FK | Пациент (опционально) |
| quantity_used | Integer | Количество |
| department | String(100) | Отделение |
| reason | String(300) | Причина |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/consumables/categories` | Список категорий | Staff |
| GET | `/consumables/items` | Список расходников (фильтр: category, search) | Staff |
| POST | `/consumables/items` | Создать расходник | Staff |
| PUT | `/consumables/items/{id}` | Обновить расходник | Staff |
| DELETE | `/consumables/items/{id}` | Удалить расходник (soft) | Staff |
| POST | `/consumables/use` | Списать расходник (уменьшает остаток) | Staff |
| GET | `/consumables/low-stock` | Расходники ниже минимума | Staff |
| GET | `/consumables/expiring` | Расходники с истекающим сроком (30 дней) | Staff |
| GET | `/consumables/usage-report` | Отчёт расхода по отделениям/позициям | Staff |
