# WhatsApp запись на приём

> Пациенты могут записаться к врачу через WhatsApp-бота. Бот понимает команды на русском языке, предлагает свободные слоты и подтверждает запись.

---

## Обзор

Интеграция с WhatsApp Business API позволяет пациентам:

1. **Записаться к врачу** — написать "Записаться к терапевту"
2. **Выбрать слот** — ответить номером из предложенного списка
3. **Проверить статус** — написать "Статус" или "Мои записи"
4. **Отменить запись** — написать "Отмена"

### Поддерживаемые специальности

терапевт, кардиолог, невролог, хирург, педиатр, окулист, ЛОР, гинеколог, дерматолог, уролог

```
Пациент: Записаться к терапевту

     1. Иванов П.С. — 23.04.2026 09:00
     2. Иванов П.С. — 23.04.2026 10:00
     3. Петрова А.В. — 23.04.2026 14:00
     Отправьте номер слота для подтверждения записи.

Пациент: 1
Бот: Запись подтверждена!
     Пациент: Уметов Асан Бакирович
     Вам придёт напоминание за 1 час до приёма.
```

## API эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| `POST` | `/api/v1/whatsapp/webhook` | Получить сообщение из WhatsApp и вернуть ответ |
| `GET` | `/api/v1/whatsapp/booking-status/{phone}` | Проверить статус записей по номеру телефона |

### Webhook payload

```json
POST /api/v1/whatsapp/webhook
{
  "phone": "+996555123456",
  "message": "Записаться к терапевту",
  "clinic_id": "550e8400-..."  // опционально
}
```

### Webhook response

```json
{
  "reply": "Доступные слоты для записи к специалисту (therapist):\n1. ..."
}
```
