# Календарь расписания (drag-and-drop)

Недельный календарь приёмов с возможностью перетаскивания (drag-and-drop) для переноса времени.

## Компоненты

### Backend API
- `GET /schedule/calendar/events` — приёмы как события календаря (фильтр по врачу и дате)
- `PUT /schedule/calendar/{id}/reschedule` — перенос приёма (new_date, new_time)
- `GET /schedule/calendar/doctors` — список врачей клиники

### Frontend
- Недельная сетка: 7 колонок (Пн-Вс), строки = 30-мин слоты (8:00–18:00)
- Навигация по неделям (пред / текущая / след)
- Фильтр по врачу (dropdown)
- Drag-and-drop через HTML5 API (onDragStart, onDrop, onDragOver)
- Цвета по статусам: SCHEDULED=синий, COMPLETED=зелёный, CANCELLED=серый, NO_SHOW=красный
- Мобильная версия: отображение одного дня с переключателем

- Frontend API: `frontend/src/features/schedule/calendar-api.ts`
