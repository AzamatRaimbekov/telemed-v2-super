# Портал пациента — новые фичи

Расширения личного кабинета пациента, добавленные в апреле 2026.

## 1. Онбординг (приветственный тур)

При первом входе пациент видит 4-шаговый тур:
1. "Добро пожаловать" — обзор кабинета
2. "Запись к врачу" — как записаться
3. "Результаты анализов" — где смотреть
4. "Голосовой ассистент" — как использовать

Показывается один раз, сохраняется в `localStorage.portal_onboarding_completed`.

**Файл:** `frontend/src/features/portal/onboarding.tsx`

## 2. Тёмная тема

Переключатель светлая/тёмная тема в header портала. Сохраняется в `localStorage.portal_theme`. Добавляет/убирает `class="dark"` на `<html>`.

**Файл:** `frontend/src/features/portal/theme-toggle.tsx`

## 3. Оценка врачей

Пациент оценивает врача (1-5 звёзд + комментарий) после приёма.

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/portal/ratings` | Оставить оценку |
| GET | `/api/v1/portal/ratings/my` | Мои оценки |
| GET | `/api/v1/portal/ratings/doctors/{id}` | Рейтинг врача |

**Модель:** `backend/app/models/rating.py` — DoctorRating (patient_id, doctor_id, rating 1-5, comment)
**Маршрут:** `backend/app/api/v1/routes/portal_ratings.py`
**Страница:** `frontend/src/routes/portal/_portal/ratings.tsx`

## 4. Семейный аккаунт

Пациент может связать аккаунты членов семьи (дети, родители, супруг) и управлять их записями.

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/portal/family/link` | Связать аккаунт |
| GET | `/api/v1/portal/family/members` | Список членов семьи |
| DELETE | `/api/v1/portal/family/{link_id}` | Отвязать |

**Модель:** `backend/app/models/family_link.py` — FamilyLink (primary_patient_id, linked_patient_id, relationship)
**Маршрут:** `backend/app/api/v1/routes/portal_family.py`
**Страница:** `frontend/src/routes/portal/_portal/family.tsx`

## 5. Напоминания о лекарствах

Пациент создаёт напоминания: название лекарства, дозировка, время приёма. Видит расписание на сегодня.

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/portal/reminders` | Создать напоминание |
| GET | `/api/v1/portal/reminders` | Список активных |
| GET | `/api/v1/portal/reminders/today` | Расписание на сегодня |
| PUT | `/api/v1/portal/reminders/{id}` | Обновить |
| DELETE | `/api/v1/portal/reminders/{id}` | Удалить |

**Модель:** `backend/app/models/medication_reminder.py` — MedicationReminder (medication_name, dosage, frequency, times JSON, start/end_date)
**Маршрут:** `backend/app/api/v1/routes/portal_reminders.py`
**Страница:** `frontend/src/routes/portal/_portal/reminders.tsx`

## 6. AI Чат (медицинский помощник)

Пациент задаёт вопросы о своих анализах и здоровье. AI (Gemini/DeepSeek) отвечает простым языком, учитывая данные пациента (диагнозы, анализы, назначения).

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/portal/ai-chat` | Отправить вопрос |

Системный промпт: "Вы медицинский ассистент. Объясняете простым языком. НЕ ставите диагнозы. Всегда рекомендуете обратиться к врачу."

**Маршрут:** `backend/app/api/v1/routes/portal_ai_chat.py`
**Страница:** `frontend/src/routes/portal/_portal/ai-chat.tsx`

## 7. Информация о клинике

Контактные данные, адрес, часы работы, парковка, этажи.

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/portal/clinic-info` | Информация о клинике |

**Маршрут:** `backend/app/api/v1/routes/portal_clinic_info.py`
**Страница:** `frontend/src/routes/portal/_portal/clinic-info.tsx`
