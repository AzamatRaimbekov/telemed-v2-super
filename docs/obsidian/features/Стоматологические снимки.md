# Стоматологические снимки (Dental Images)

> Загрузка, хранение и просмотр стоматологических снимков: рентген, ОПТГ, КЛКТ, интраоральные фото, до/после.

---

## Архитектура

### Backend
- **Модель:** `backend/app/models/dental_image.py`
  - `DentalImage` — снимок (тип, привязка к зубам, URL, пара до/после)
  - `DentalImageType` — enum типов (periapical, panoramic, bitewing, cephalometric, ct_3d, photo_intraoral, photo_extraoral, photo_before_after)
- **Маршруты:** `backend/app/api/v1/routes/dental_images.py`
  - `POST /dental/images` — загрузка (multipart)
  - `GET /dental/images/patient/{patient_id}` — все снимки пациента
  - `GET /dental/images/tooth/{patient_id}/{tooth_number}` — снимки конкретного зуба
  - `GET /dental/images/{id}` — детали снимка
  - `DELETE /dental/images/{id}` — удаление
  - `GET /dental/images/before-after/{patient_id}` — пары до/после

## Типы снимков

| Тип | Описание |
|-----|----------|
| periapical | Прицельный снимок |
| panoramic | ОПТГ (панорамный) |
| bitewing | Интерпроксимальный |
| cephalometric | ТРГ (телерентгенограмма) |
| ct_3d | КЛКТ (3D томография) |
| photo_intraoral | Фото внутриротовое |
| photo_extraoral | Фото внеротовое |
| photo_before_after | Фото до/после |
