# Фото-консультация

> Пациент отправляет фото стоматологической проблемы для удалённой консультации врачом.

---

## Обзор

Пациент загружает описание проблемы и указывает срочность (обычная / срочная). Врач получает обращение и отвечает в течение 24 часов.

---

## Архитектура

### Бэкенд

- **Маршруты:** `backend/app/api/v1/routes/patient_photo_consult.py`

---

## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/portal/photo-consult/submit` | Отправить обращение (Form: patient_id, description, urgency) |
| GET | `/api/v1/portal/photo-consult/my-consults` | Мои консультации |
