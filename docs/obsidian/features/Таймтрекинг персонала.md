# Таймтрекинг персонала (Time Tracking)

## Описание

Учёт рабочего времени сотрудников клиники. Приход/уход (clock in/out), автоматический расчёт отработанных часов с учётом перерывов, контроль переработок, отчёты по сотрудникам и отделениям.

## Модели БД

### `time_entries`
| Поле | Тип | Описание |
|------|-----|----------|
| user_id | UUID FK | Сотрудник |
| clock_in | DateTime(tz) | Время прихода |
| clock_out | DateTime(tz) | Время ухода |
| hours_worked | Float | Отработано часов (нетто) |
| break_minutes | Float | Перерыв (минуты) |
| overtime_hours | Float | Переработка (часов, свыше 8ч) |
| notes | Text | Заметки |
| department | String(100) | Отделение |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| POST | `/time-tracking/clock-in` | Отметить приход | Staff |
| POST | `/time-tracking/clock-out` | Отметить уход (расчёт часов) | Staff |
| GET | `/time-tracking/my` | Мои записи (фильтр: from_date, to_date) | Staff |
| GET | `/time-tracking/status` | Текущий статус (на смене / нет) | Staff |
| GET | `/time-tracking/report` | Отчёт по сотрудникам/отделениям (admin) | Staff |
| GET | `/time-tracking/overtime` | Алерты переработок (порог, период) | Staff |
