# Лаборатория

Управление лабораторными исследованиями.

## Процесс

```
Врач создаёт LabOrder → Лаборант выполняет → LabResult →
    Врач утверждает (approve) → Пациент видит в портале
```

## Модели

- **LabTestCatalog** — каталог доступных анализов (нормы, единицы)
- **LabOrder** — заказ на анализ (пациент, врач, тесты)
- **LabResult** — результат (значение, флаг normal/abnormal, комментарий)
