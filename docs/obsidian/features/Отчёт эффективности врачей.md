# Отчёт эффективности врачей

Аналитический отчёт по продуктивности врачей: количество приёмов, процент завершения, среднее количество в день, рейтинг.

## Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/reports/doctors/efficiency` | Метрики эффективности по каждому врачу за период |
| GET | `/api/v1/reports/doctors/ranking` | Топ-10 врачей по количеству приёмов за текущий месяц |

## Метрики

- **total_appointments** — общее количество приёмов за период
- **completed_appointments** — завершённые приёмы
- **completion_rate** — процент завершения (%)
- **avg_per_day** — среднее количество приёмов в день
- **specialization** — специализация врача

## Параметры

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| date_from | date | -30 дней | Начало периода |
| date_to | date | сегодня | Конец периода |
