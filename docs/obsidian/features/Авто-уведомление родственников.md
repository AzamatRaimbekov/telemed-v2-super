# Авто-уведомление родственников (Relative Notify)

## Описание

Автоматическая отправка SMS-уведомлений родственникам/опекунам пациента при поступлении (admission) и выписке (discharge). Система находит всех привязанных опекунов (PatientGuardian) и отправляет им сообщения через NotificationDispatcher.

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| POST | `/api/v1/relative-notify/admission` | Уведомить о поступлении |
| POST | `/api/v1/relative-notify/discharge` | Уведомить о выписке |

## Тело запроса

```json
{
  "patient_id": "uuid"
}
```

## Ответ

```json
{
  "notified": 2,
  "total_guardians": 3
}
```

## Сервис

**Файл:** `backend/app/services/relative_notifier.py`

- `RelativeNotifierService.notify_admission()` — уведомление при поступлении
- `RelativeNotifierService.notify_discharge()` — уведомление при выписке
- Использует `PatientGuardian.guardian.phone` для получения телефона
- Отправка через `NotificationDispatcher` (канал SMS)

## Шаблоны сообщений

**Поступление:**
> Уважаемый(ая)! Пациент {ФИО} поступил(а) в клинику. Для информации обратитесь в регистратуру.

**Выписка:**
> Пациент {ФИО} выписан(а) из клиники. Подробности в регистратуре.

## Связи

- Модели: `Patient`, `PatientGuardian`, `User` (guardian)
- Использует: [[Уведомления (SMS, WhatsApp, Telegram)]] (NotificationDispatcher)
- Роут: `backend/app/api/v1/routes/relative_notify.py`
