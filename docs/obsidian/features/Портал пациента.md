# Портал пациента

Личный кабинет пациента с отдельной авторизацией и [[Голосовой ассистент|голосовым управлением]].

## Авторизация

- **Вход:** телефон + пароль (не email)
- **JWT:** отдельный токен с `patient_id`
- **API клиент:** `frontend/src/lib/portal-api-client.ts`

## Модули портала

### Dashboard
Сводка: предстоящие приёмы, последние результаты, активные планы лечения, статус восстановления.

### Медицинская карта
Просмотр данных медкарты: диагнозы, аллергии, группа крови, хронические заболевания.

### История болезни (36KB)
Хронология всех медицинских событий: визиты, назначения, операции.

### Результаты анализов
Только **утверждённые** (approved) результаты лабораторных исследований.

### Запись на приём (37KB)
- Выбор врача и специальности
- Доступные слоты из расписания
- Подтверждение записи

### Упражнения и реабилитация
- Каталог упражнений (22KB)
- Сессия с MediaPipe видео (43KB) — определение позы в реальном времени
- Прогресс и статистика (18KB)

### Восстановление (22KB)
Цели восстановления с визуализацией прогресса по доменам.

### Лечение (23KB)
Просмотр активных планов лечения и назначений.

### Сообщения (14KB)
Чат с лечащим врачом через телемедицинскую сессию.

### Биллинг (20KB)
Просмотр и оплата счетов.

### Профиль (25KB)
Редактирование контактных данных, смена пароля.

## Backend

- **Маршрут:** `backend/app/api/v1/routes/portal.py` (25KB)
