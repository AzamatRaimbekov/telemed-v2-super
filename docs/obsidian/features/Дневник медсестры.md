# Дневник медсестры (Nurse Daily Diary)

## Описание

Ежедневный дневник наблюдения медсестры за пациентом. Фиксация витальных показателей, общего состояния, жалоб, выполненных процедур и выданных лекарств по сменам (день, вечер, ночь).

## Модели БД

### `nurse_diary_entries`
| Поле | Тип | Описание |
|------|-----|----------|
| patient_id | UUID FK | Пациент |
| nurse_id | UUID FK | Медсестра |
| entry_date | Date | Дата записи |
| shift | String(20) | Смена: day / evening / night |
| general_condition | String(50) | Общее состояние: satisfactory / moderate / severe / critical |
| consciousness | String(50) | Сознание: clear / confused / unconscious |
| temperature | String(10) | Температура |
| blood_pressure | String(20) | Артериальное давление |
| pulse | String(10) | Пульс |
| respiratory_rate | String(10) | Частота дыхания |
| complaints | Text | Жалобы пациента |
| procedures_done | Text | Выполненные процедуры |
| medications_given | JSON | Выданные лекарства [{name, dose, time}] |
| diet | String(100) | Диета |
| notes | Text | Примечания |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| POST | `/nurse-diary/` | Создать запись | Staff |
| GET | `/nurse-diary/patient/{patient_id}` | Записи по пациенту (фильтр по датам) | Staff |
| GET | `/nurse-diary/my-shift` | Записи текущей медсестры за сегодня | Staff |
| GET | `/nurse-diary/patient/{patient_id}/latest` | Последняя запись по пациенту | Staff |
| GET | `/nurse-diary/{id}` | Детали записи | Staff |
| PATCH | `/nurse-diary/{id}` | Обновить запись | Staff |
