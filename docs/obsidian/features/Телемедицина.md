# Телемедицина

Видеоконсультации врач-пациент через Daily.co.

## Возможности

- Создание видеосессии (Daily.co room)
- Подключение врача и пациента
- Текстовый чат во время сессии
- История сообщений

## Модели

- **TelemedicineSession** — сессия видеозвонка (Daily.co room URL, статус, участники)
- **Message** — сообщения чата
