# Экспорт отчётов

Экспорт данных в Excel для аналитики и отчётности.

## Доступные экспорты

| Тип | Формат | Эндпоинт |
|-----|--------|----------|
| Пациенты | Excel (.xlsx) | `GET /export/patients/excel` |
| Приёмы | Excel (.xlsx) | `GET /export/appointments/excel` |
| Платежи | Excel (.xlsx) | `GET /export/payments/excel` |

## Формат Excel

- Заголовок отчёта + дата генерации
- Стилизованные заголовки (синий фон, белый текст)
- Автоширина колонок
- Тонкие границы ячеек
