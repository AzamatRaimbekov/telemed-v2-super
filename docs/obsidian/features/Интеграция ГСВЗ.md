# Интеграция с ГСВЗ (Государственная система здравоохранения КР)

## Описание

Placeholder-интеграция с государственной системой здравоохранения Кыргызстана. В демо-режиме возвращает заглушки; при настройке `GSVZ_API_URL` и `GSVZ_API_KEY` делает реальные HTTP-запросы.

## Функции

1. **Проверка страховки** — по ИНН пациента
2. **Отправка отчёта о визите** — для гос. статистики
3. **Обновления МКБ-10** — получение последних версий классификатора
4. **Статус подключения** — проверка связи с ГСВЗ

## Конфигурация

Переменные окружения:

| Переменная | Описание |
|-----------|----------|
| `GSVZ_API_URL` | Базовый URL API ГСВЗ |
| `GSVZ_API_KEY` | Ключ API для авторизации |

Без этих переменных система работает в демо-режиме.

## API эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/gsvz/verify-insurance/{inn}` | Проверка страховки по ИНН |
| POST | `/gsvz/submit-visit` | Отправка отчёта о визите |
| GET | `/gsvz/icd10-updates` | Получение обновлений МКБ-10 |
| GET | `/gsvz/status` | Статус подключения к ГСВЗ |

## Файлы

- **Роуты:** `backend/app/api/v1/routes/gsvz.py`
- **Конфиг:** `backend/app/core/config.py` (GSVZ_API_URL, GSVZ_API_KEY)
