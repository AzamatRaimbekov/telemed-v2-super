# Выписка онлайн (Online Discharge)

> Электронная выписка пациента. Врач заполняет диагноз, резюме лечения и рекомендации. Автоматически деактивирует браслет пациента и отправляет данные в портал.

---

## Обзор

При выписке:
1. Врач заполняет форму (диагноз, лечение, рекомендации, лекарства, дата повторного визита)
2. Система деактивирует все активные браслеты пациента (статус → `discharged`)
3. Уведомление отправляется в портал пациента
4. Сводка доступна для печати

---

## Архитектура

### Бэкенд

- **Маршруты:** `backend/app/api/v1/routes/discharge.py`
- Использует модели `Patient`, `PatientWristband`
- Новых моделей не требуется

---

## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/discharge/` | Создать выписку |
| GET | `/api/v1/discharge/{patient_id}/summary` | Сводка для печати/портала |

### Тело запроса POST

```json
{
  "patient_id": "uuid",
  "diagnosis": "Острый инсульт",
  "treatment_summary": "Тромболизис, наблюдение 7 дней",
  "recommendations": "Приём препаратов, контроль давления",
  "follow_up_date": "2026-05-15",
  "medications": [
    { "name": "Аспирин", "dosage": "100мг", "frequency": "1 раз/день" }
  ]
}
```

### Ответ

```json
{
  "status": "discharged",
  "patient_name": "Уметов Асан",
  "discharge": { "diagnosis": "...", "treatment": "...", "recommendations": "...", "follow_up": "...", "medications": [...] },
  "wristbands_deactivated": 1,
  "portal_notification": "sent"
}
```
