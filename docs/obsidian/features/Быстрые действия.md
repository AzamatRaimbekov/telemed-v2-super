# Быстрые действия (FAB)

Плавающая кнопка в правом нижнем углу для быстрого доступа к частым действиям.

## Действия
- Новый пациент -> /patients/new
- Новый приём -> /schedule
- Направление -> /referrals
- Рецепт -> /document-templates
- Анализы -> /laboratory
- AI Суммаризация -> /visit-summaries

**Файл:** `frontend/src/components/shared/quick-actions.tsx`
