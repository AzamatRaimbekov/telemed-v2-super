# Экран у койки (Bedside Dashboard)

> Публичный full-screen дисплей для планшета у кровати пациента. Показывает расписание на день, информацию о пациенте и кнопку вызова медсестры. Работает по UID браслета без авторизации.

---

## Обзор

Планшет у койки отображает:
- ФИО пациента и UID браслета
- Расписание на сегодня (лекарства, процедуры, обход врача, питание)
- Текущее время с подсветкой активного пункта расписания
- Кнопку вызова медсестры
- Сообщение от врача (если есть)
- Автоматический тёмный режим в ночное время (21:00-07:00)

---

## Архитектура

### Бэкенд

- **Маршрут:** `backend/app/api/v1/routes/bedside.py`
- Публичный endpoint, без авторизации
- Агрегирует данные из существующих моделей (Patient, PatientWristband)

### Фронтенд

- **Страница:** `frontend/src/routes/bedside.tsx`
- Публичный маршрут `/bedside/{wristband_uid}` — без `_authenticated`
- Auto-refresh каждые 30 секунд

---

## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/bedside/{wristband_uid}` | Данные для экрана у койки |

### Ответ

```json
{
  "patient": { "first_name": "...", "last_name": "...", "wristband_uid": "MC-XXXXXX" },
  "today_schedule": [
    { "time": "08:00", "type": "medication", "description": "Приём лекарств" }
  ],
  "nurse_call_available": true,
  "message_from_doctor": null
}
```

---

## UI компоненты

- Крупные шрифты, читаемые с кровати
- Иконки по типу события (Pill, Stethoscope, UtensilsCrossed, Activity)
- Подсветка текущего пункта расписания
- Анимация появления элементов
- Тёмный/светлый режим автоматически
