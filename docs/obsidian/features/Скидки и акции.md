# Скидки и акции (Promotions)

## Описание

Управление скидками и промо-акциями клиники. Поддержка процентных и фиксированных скидок, промокодов, лимитов использования, дат действия, привязки к конкретным услугам. Валидация промокода с расчётом итоговой суммы.

## Модели БД

### `promotions`
| Поле | Тип | Описание |
|------|-----|----------|
| name | String(200) | Название акции |
| code | String(50) | Промокод (уникальный, опционально) |
| description | Text | Описание |
| discount_type | Enum | percent / fixed |
| discount_value | Float | Размер скидки (% или сумма) |
| min_amount | Float | Минимальная сумма заказа |
| max_uses | Integer | Макс. кол-во использований |
| used_count | Integer | Сколько раз использовано |
| valid_from | Date | Начало действия |
| valid_until | Date | Конец действия |
| applicable_services | JSON | Применимые услуги |
| is_active | Boolean | Активна ли акция |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/promotions` | Список акций (фильтр: active_only) | Staff |
| POST | `/promotions` | Создать акцию | Staff |
| PUT | `/promotions/{id}` | Обновить акцию | Staff |
| DELETE | `/promotions/{id}` | Удалить акцию (soft) | Staff |
| POST | `/promotions/validate` | Проверить промокод и рассчитать скидку | Staff |
| GET | `/promotions/active` | Активные акции (в рамках дат) | Staff |
