# Скелетоны загрузки

> Универсальные скелетоны для отображения во время загрузки данных на всех страницах.

---

## Обзор

Набор компонентов-скелетонов, которые показываются вместо контента во время загрузки. Используют CSS `animate-pulse` и CSS-переменные дизайн-системы.

## Компоненты

### `PageSkeleton`

Полный скелетон страницы: заголовок, статистика (4 карточки), список (5 строк).

### `CardSkeleton`

Скелетон отдельной карточки: аватар, заголовок, описание.

### `TableSkeleton`

Скелетон таблицы с заголовком и настраиваемым количеством строк.

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|-------------|----------|
| `rows` | number | 5 | Количество строк |

## Использование

```tsx
import { PageSkeleton, CardSkeleton, TableSkeleton } from "@/components/shared/page-skeleton";

// Полная страница
if (isLoading) return <PageSkeleton />;

// Карточка
if (isLoading) return <CardSkeleton />;

// Таблица с 10 строками
if (isLoading) return <TableSkeleton rows={10} />;
```
