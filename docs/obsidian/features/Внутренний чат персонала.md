# Внутренний чат персонала

## Описание

Внутренний мессенджер для сотрудников клиники. Групповые каналы (Общий, Врачи, Медсёстры) и личные сообщения. Автообновление каждые 5 секунд.

## Модели БД

### `staff_messages`
| Поле | Тип | Описание |
|------|-----|----------|
| sender_id | UUID FK | Отправитель |
| recipient_id | UUID FK | Получатель (null = групповое) |
| channel | String(100) | Канал: general, doctors, nurses, dm:{id1}:{id2} |
| body | Text | Текст сообщения |
| is_read | Boolean | Прочитано |

## API эндпоинты

| Метод | Путь | Описание | Авторизация |
|-------|------|----------|-------------|
| GET | `/chat/channels` | Список каналов | Staff |
| GET | `/chat/messages` | Сообщения канала | Staff |
| POST | `/chat/messages` | Отправить сообщение | Staff |
| PATCH | `/chat/messages/{id}/read` | Отметить прочитанным | Staff |
| GET | `/chat/unread-count` | Количество непрочитанных | Staff |
