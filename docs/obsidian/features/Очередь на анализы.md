# Электронная очередь на анализы

Очередь пациентов для забора анализов с публичным табло для холла лаборатории.

## Компоненты

### Управление (для лаборанта)
- Добавление пациента в очередь
- Вызов следующего к окну забора
- Завершение / пропуск пациента
- Привязка к направлению на анализ (lab_order_id)

### Публичное табло (без авторизации)
- Отображение текущего и ожидающих номеров
- Группировка по окнам забора
- Автообновление для ТВ-экрана

## Статусы очереди

| Статус | Описание |
|--------|----------|
| `waiting` | Ожидает вызова |
| `called` | Вызван к окну |
| `collecting` | Идёт забор |
| `completed` | Завершён |
| `skipped` | Пропущен |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/lab-queue/add` | Добавить в очередь |
| POST | `/api/v1/lab-queue/call-next?window=1` | Вызвать следующего к окну |
| POST | `/api/v1/lab-queue/{id}/complete` | Завершить (анализ взят) |
| POST | `/api/v1/lab-queue/{id}/skip` | Пропустить |
| GET | `/api/v1/lab-queue/today` | Очередь на сегодня |
| GET | `/api/v1/lab-queue/display?clinic_id=...` | Публичное табло (без авторизации) |
