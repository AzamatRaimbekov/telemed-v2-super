# Дашборд главврача

Единый экран с KPI клиники для руководителя.

## KPI показатели

| Показатель | Описание |
|-----------|----------|
| Пациенты | Всего, новых за неделю/месяц |
| Приёмы | Сегодня, за неделю, % выполнения |
| Палаты | Загрузка %, занято/всего |
| Доход | За месяц, рост к прошлому |
| Лаборатория | Ожидающие результаты |
| Аптека | Позиции с низким остатком |
| Персонал | Врачей, медсестёр, всего |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/chief-dashboard/` | Все KPI одним запросом |

Автообновление каждые 30 секунд.
