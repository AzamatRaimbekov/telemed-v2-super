# Реферальная программа

> Пациент получает персональный реферальный код. Друг использует код и получает скидку, а реферер — бонус.

---

## Обзор

Каждый пациент может сгенерировать уникальный реферальный код (формат `MED-XXXXXXXX`). При использовании кода новый пациент получает скидку (по умолчанию 10%), а владелец кода — бонус (5%). Защита от самоиспользования и повторного применения.

---

## Архитектура

### Модели данных

**Таблица:** `referral_codes`

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | UUID | PK |
| `clinic_id` | UUID | Мультитенант |
| `patient_id` | UUID | FK -> patients |
| `code` | String(20) | Уникальный код |
| `discount_percent` | Float | Скидка для нового пациента (%) |
| `referrer_bonus_percent` | Float | Бонус для реферера (%) |
| `uses_count` | Integer | Сколько раз использован |
| `max_uses` | Integer? | Лимит использований |
| `is_active` | Boolean | Активен ли |

**Таблица:** `referral_uses`

| Поле | Тип | Описание |
|------|-----|----------|
| `id` | UUID | PK |
| `clinic_id` | UUID | Мультитенант |
| `code_id` | UUID | FK -> referral_codes |
| `referred_patient_id` | UUID | FK -> patients |
| `discount_amount` | Float | Предоставленная скидка |
| `bonus_amount` | Float | Начисленный бонус |

### Бэкенд

- **Модели:** `backend/app/models/referral_program.py`
- **Маршруты:** `backend/app/api/v1/routes/referral_program.py`

## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/referral/my-code` | Получить/создать реферальный код |
| POST | `/api/v1/referral/use` | Использовать реферальный код |
| GET | `/api/v1/referral/my-referrals` | Список использований моего кода |
| GET | `/api/v1/referral/my-bonuses` | Итого заработанных бонусов |

---

## Бизнес-логика

1. Пациент открывает страницу рефералов → GET `/my-code` создаёт код при первом обращении
2. Пациент делится кодом с другом
3. Друг вводит код → POST `/use` валидирует и применяет скидку
4. Защита: нельзя использовать свой код, нельзя использовать дважды
5. Владелец кода видит статистику через `/my-referrals` и `/my-bonuses`
