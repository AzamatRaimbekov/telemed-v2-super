# Горячие клавиши

Навигация по системе с клавиатуры.

## Комбинации (Alt + клавиша)
| Клавиша | Действие |
|---------|----------|
| Alt+D | Панель управления |
| Alt+P | Пациенты |
| Alt+S | Расписание |
| Alt+T | Задачи |
| Alt+C | Чат |
| Alt+R | Отчёты |
| Alt+Q | Очередь |

Не срабатывают когда фокус в input/textarea.

**Файл:** `frontend/src/hooks/useHotkeys.ts`
