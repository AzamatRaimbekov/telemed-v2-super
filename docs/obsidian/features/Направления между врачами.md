# Направления между врачами

Система направлений пациентов от одного врача к другому с приоритетами и отслеживанием статуса.

## Возможности

- Создание направления с указанием специальности и приоритета
- Входящие / исходящие направления для каждого врача
- Принятие, отклонение, завершение направления
- Фильтрация по статусу, приоритету, врачу
- Указание диагноза и заметок

## Приоритеты

| Приоритет | Описание |
|-----------|----------|
| `routine` | Плановый |
| `urgent` | Срочный |
| `emergency` | Экстренный |

## Статусы

| Статус | Описание |
|--------|----------|
| `pending` | Ожидает рассмотрения |
| `accepted` | Принято |
| `declined` | Отклонено |
| `completed` | Завершено |

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/v1/referrals/` | Создать направление |
| GET | `/api/v1/referrals/` | Список (фильтры: from_doctor, to_doctor, status, priority) |
| GET | `/api/v1/referrals/{id}` | Детали направления |
| PATCH | `/api/v1/referrals/{id}/accept` | Принять |
| PATCH | `/api/v1/referrals/{id}/decline` | Отклонить с причиной |
| PATCH | `/api/v1/referrals/{id}/complete` | Завершить |
| GET | `/api/v1/referrals/incoming` | Мои входящие направления |
| GET | `/api/v1/referrals/outgoing` | Мои исходящие направления |
