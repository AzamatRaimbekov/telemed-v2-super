# Видео-инструкции

> Библиотека обучающих видео для пациентов: уход после лечения, гигиена, ортодонтия.

---

## Обзор

Статическая библиотека видео-инструкций с категориями. Пациент фильтрует по категории, смотрит описание и длительность. Категории: После лечения (4 видео), Ортодонтия (2), Гигиена (2).

---

## Архитектура

### Бэкенд

- **Маршруты:** `backend/app/api/v1/routes/video_instructions.py`
- Данные хранятся в коде (статический список), без модели БД

### Фронтенд

- **Страница:** `frontend/src/routes/portal/_portal/videos.tsx`

---

## API Эндпоинты

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/video-instructions/` | Список видео (опционально ?category=) |
| GET | `/api/v1/video-instructions/categories` | Категории видео |
| GET | `/api/v1/video-instructions/{video_id}` | Детали видео |
