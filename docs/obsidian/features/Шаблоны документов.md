# Шаблоны документов

Предзаполненные медицинские шаблоны с автоподстановкой данных пациента.

## Категории шаблонов

| Категория | Описание |
|-----------|----------|
| `prescription` | Рецепт |
| `discharge` | Выписка |
| `referral` | Направление |
| `certificate` | Справка |
| `consent` | Согласие |
| `lab_order` | Направление на анализы |

## Шаблоны по умолчанию (seed)

1. **Рецепт** — ФИО, дата, лекарства, врач, подпись
2. **Выписка** — даты госпитализации, диагноз, лечение, рекомендации
3. **Направление** — от кого, куда, пациент, причина
4. **Справка** — ФИО, что лечился с/по даты

## Jinja2 переменные

```
{{ patient.last_name }}, {{ patient.first_name }}
{{ patient.date_of_birth }}, {{ patient.inn }}
{{ doctor.name }}, {{ doctor.specialty }}
{{ visit.date }}, {{ visit.diagnosis }}
{{ clinic.name }}, {{ clinic.address }}
```

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/v1/document-templates/` | Список шаблонов |
| POST | `/api/v1/document-templates/` | Создать шаблон |
| GET | `/api/v1/document-templates/{id}` | Получить шаблон |
| PUT | `/api/v1/document-templates/{id}` | Обновить |
| DELETE | `/api/v1/document-templates/{id}` | Удалить |
| POST | `/api/v1/document-templates/{id}/render` | Рендер с данными |
