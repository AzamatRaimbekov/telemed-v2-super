# Обзор проекта MedCore KG

## Назначение

MedCore KG — это телемедицинская SaaS-платформа, разработанная для клиник и больниц Кыргызстана. Система охватывает полный цикл медицинского обслуживания: от регистрации пациента до реабилитации.

## Основные возможности

### Для врачей и персонала
- Электронные медицинские карты (EHR)
- Назначение лечения и рецептов
- Видеоконсультации через Daily.co
- Расписание приёмов
- Аналитика и отчёты
- Аудит всех действий

### Для пациентов ([[Портал пациента]])
- Личный кабинет с доступом через телефон
- Просмотр медкарты и результатов анализов
- Запись на приём
- Домашние упражнения с MediaPipe
- [[Голосовой ассистент]] для навигации
- Оплата счетов

### Для администрации
- [[Аутентификация и RBAC]] — управление ролями (9+ ролей)
- [[Управление зданием (BMS)]] — IoT датчики и оборудование
- [[Мониторинг пациентов]] — жизненные показатели в реальном времени
- [[Фармация]] — управление аптекой
- [[Биллинг]] — финансовый учёт

## Архитектурные принципы

1. **Мультитенантность** — каждая клиника изолирована через `TenantMixin`
2. **Repository Pattern** — разделение доступа к данным и бизнес-логики
3. **JWT + Redis** — безопасная аутентификация с чёрным списком токенов
4. **WebSocket** — реальное время для мониторинга и BMS
5. **Celery** — асинхронные задачи (email, уведомления)
6. **File-based Routing** — TanStack Router на фронтенде
7. **Idempotent Seeds** — безопасный повторный запуск seed-скриптов
