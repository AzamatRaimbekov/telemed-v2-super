# Добавление нового модуля

Пошаговое руководство по созданию нового функционального модуля в проекте.

## 1. Backend

### 1.1 Модель (`backend/app/models/`)
```python
# backend/app/models/new_module.py
from app.models.base import Base, BaseMixin, TenantMixin

class NewEntity(Base, BaseMixin, TenantMixin):
    __tablename__ = "new_entities"

    name = Column(String, nullable=False)
    # ... поля
```

Добавить import в `backend/app/models/__init__.py`.

### 1.2 Схема (`backend/app/schemas/`)
```python
# backend/app/schemas/new_module.py
from pydantic import BaseModel

class NewEntityCreate(BaseModel):
    name: str

class NewEntityResponse(BaseModel):
    id: int
    name: str

    class Config:
        from_attributes = True
```

### 1.3 Сервис (`backend/app/services/`)
```python
# backend/app/services/new_module.py
from sqlalchemy.ext.asyncio import AsyncSession

class NewModuleService:
    def __init__(self, db: AsyncSession):
        self.db = db

    async def create(self, data: NewEntityCreate) -> NewEntity:
        ...
```

### 1.4 Маршрут (`backend/app/api/v1/routes/`)
```python
# backend/app/api/v1/routes/new_module.py
from fastapi import APIRouter, Depends

router = APIRouter(prefix="/new-module", tags=["new-module"])

@router.get("/")
async def list_entities(...):
    ...
```

Зарегистрировать в `backend/app/api/v1/__init__.py`.

### 1.5 Миграция
```bash
cd backend
alembic revision --autogenerate -m "add new_entities table"
alembic upgrade head
```

### 1.6 Seed (опционально)
```python
# backend/seed_new_module.py
```
Добавить вызов в `backend/app/main.py` (lifespan).
