# Сервисы

## Voice Assistant (подмодуль)

Расположен в `backend/app/services/voice_assistant/`:

```
voice_assistant/
├── service.py          # Оркестрация голосовых запросов
├── action_executor.py  # Выполнение голосовых команд
├── context_builder.py  # Построение контекста для AI
└── ai_providers/       # Gemini, DeepSeek, Whisper
```
