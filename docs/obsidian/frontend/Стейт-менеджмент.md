# Стейт-менеджмент

## Zustand Stores

### auth-store.ts

```
src/stores/auth-store.ts
```

Глобальное состояние аутентификации:

- `token` — JWT access token
- `user` — текущий пользователь (id, email, role)
- `clinic_id` — ID текущей клиники
- `isAuthenticated` — флаг авторизации
- `login()` — сохранение токена
- `logout()` — очистка состояния

## TanStack Query

Серверное состояние управляется через TanStack Query 5.62:

- **Query Client:** `src/lib/query-client.ts`
- **Feature API:** каждый модуль в `features/` содержит `api.ts` с хуками:
  - `usePatients()`, `usePatient(id)`
  - `useLabOrders()`, `useLabResults()`
  - `usePrescriptions()`
  - и т.д.

## WebSocket состояние

Для реального времени используются хуки:

- `features/monitoring/use-monitoring-ws.ts` — данные датчиков
- `features/infrastructure/use-bms-ws.ts` — данные BMS
