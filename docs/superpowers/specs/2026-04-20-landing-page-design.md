## Концепция

Премиальный лендинг с 3D-элементами для привлечения медицинских учреждений. Посыл: "Единая платформа — замените 10 систем одной". Тёмная тема на основе существующей дизайн-системы проекта.

## Визуальный стиль

- **Фон:** `#0B0F14` (dark mode из дизайн-системы)
- **Primary:** `#BDEDE0` (мятный), deep: `#7ECDB8`
- **Secondary:** `#7E78D2` (фиолетовый), deep: `#5B54B0`
- **Surface:** `#141921`, elevated: `#1A202C`
- **Text:** `#F1F5F9` primary, `#94A3B8` secondary, `#64748B` tertiary
- **Success:** `#10B981`, Warning: `#F59E0B`, Danger: `#EF4444`
- **Border:** `#1E293B`
- **Шрифт:** Plus Jakarta Sans (уже подключён)
- **Радиус:** `0.75rem` (из дизайн-системы)
- **Эффекты:** glassmorphism карточки, radial glow, gradient акценты

## 3D-технология

**React Three Fiber (R3F) + @react-three/drei**

Три 3D-сцены:
1. **Hero** — орбита модулей вокруг центральной сферы
2. **Портал пациента** — 3D модель телефона
3. **IoT мониторинг** — интерактивная 3D палата с датчиками

Подход к загрузке: lazy-load каждый Canvas через `React.lazy` + `Suspense`. Fallback — статичная CSS версия для слабых устройств. `useReducedMotion` для отключения анимаций по настройкам ОС.

## Анимации

- **Scroll-driven:** Framer Motion `useScroll` + `useTransform` для появления секций
- **3D:** R3F `useFrame` для вращения орбиты, парения телефона, пульсации датчиков
- **Hover:** Framer Motion для карточек модулей (scale + glow)
- **Transition:** секции появляются снизу с fade при скролле

### Маршрутизация

Текущий `index.tsx` делает redirect на `/login`. Новая схема:
- `/` → `landing.tsx` (лендинг для неавторизованных)
- `/login` → существующая страница логина
- `/dashboard` → redirect для авторизованных (без изменений)

В `landing.tsx`:
```typescript
export const Route = createFileRoute("/")({
  beforeLoad: ({ context }) => {
    if (context.auth.isAuthenticated) {
      throw redirect({ to: "/dashboard" });
    }
  },
  component: LandingPage,
});
```

### Производительность

- Lazy-load R3F Canvas через `React.lazy` + `Suspense`
- Fallback: статичная CSS-версия для каждой 3D-сцены
- `useReducedMotion` — отключение анимаций по настройкам ОС
- `IntersectionObserver` — рендер 3D только когда секция видна
- Изображения: WebP, lazy loading
- Bundle: три 3D-сцены в отдельных chunks (~50-80KB каждый)

### Responsive

- Desktop: полная 3D, все эффекты
- Tablet: упрощённые 3D (меньше полигонов), 2 колонки
- Mobile: CSS fallback вместо 3D, 1 колонка, упрощённые анимации

### Доступность

- `prefers-reduced-motion` — отключение всех анимаций
- Семантический HTML (nav, main, section, footer)
- Alt-тексты для всех визуальных элементов
- Клавиатурная навигация для табов и форм
- Контрастность текста WCAG AA на тёмном фоне
