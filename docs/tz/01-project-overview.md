# MedCore KG — Обзор проекта

SaaS система управления клиниками Кыргызстана.
16 модулей, 55+ API эндпоинтов, мультитенантность.

## Модули
1. Auth & Roles (9 ролей, JWT + Redis blacklist)
2. Patient Registration (OCR + Face Detection + Form)
3. Medical Card (EHR)
4. Treatment Plan Builder
5. Stroke Rehabilitation
6. Staff Management
7. Pharmacy & Formulary
8. Laboratory
9. Billing & Finance
10. Appointments & Scheduling
11. Notifications (WebSocket)
12. Patient Portal (/portal/*)
13. Telemedicine
14. Admin Dashboard
15. Super Admin (Multi-clinic)
16. MediaPipe Exercise Tracking
