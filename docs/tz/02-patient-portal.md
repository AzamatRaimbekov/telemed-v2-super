# ТЗ — Модуль Пациента (Patient Portal)

8 под-модулей, 38 API эндпоинтов.
Статус: РЕАЛИЗОВАН (Phase A + B)

## Под-модули
1. Регистрация и профиль — портал логин (телефон+пароль), профиль
2. Медицинская карта — показатели, диагнозы МКБ-10, документы
3. Лабораторные результаты — только approved врачом, тренды
4. Финансы и счета — сводка, инвойсы, платежи, PDF
5. Упражнения (MediaPipe) — 15 упражнений, сессии, прогресс
6. Телемедицина — Daily.co, видеозвонки
7. Сообщения — WebSocket чат с врачом
8. Запись на приём — слоты, бронирование
