# MedCore KG — Телемедицинская SaaS-платформа

> Единая платформа для управления клиниками, поликлиниками и медицинскими центрами Кыргызстана.

---

## Проблема

Медицинские учреждения Кыргызстана работают на **10+ разных программах** одновременно: Excel для учёта пациентов, бумажные медкарты, WhatsApp для коммуникации, отдельные программы для аптеки, лаборатории, бухгалтерии. Данные разрозненны, теряются, не связаны друг с другом. Нет аналитики, нет контроля, нет удобного сервиса для пациентов.

## Решение

**MedCore KG** — единая облачная платформа, которая заменяет все разрозненные инструменты **одной системой**. 16 модулей охватывают полный цикл медицинского обслуживания: от регистрации пациента до выписки, от назначения лечения до контроля оборудования здания.

---

## Модули системы (16)

### Медицина

| Модуль | Описание |
|--------|----------|
| **Электронные медкарты (EHR)** | Карточки пациентов, визиты, диагнозы по МКБ-10, витальные показатели, история болезни. Мгновенный поиск по ФИО, ИНН, телефону. |
| **Фармация** | Полное управление аптекой: каталог лекарств, рецепты врачей, складской учёт, поставщики, заказы. Контроль сроков годности и минимальных остатков. |
| **Лаборатория** | Заказы на анализы, внесение результатов, утверждение врачом. Пациент видит только утверждённые результаты. Каталог 500+ анализов. |
| **Лечение** | Структурированные планы лечения с назначениями, контроль выполнения, привязка к диагнозу. |
| **Реабилитация** | Специализированный модуль для инсульт-реабилитации: оценка по шкалам NIHSS/mRS, цели восстановления, домашние упражнения с AI-трекером через камеру (MediaPipe). |
| **Аналитика** | Дашборды для руководства: загрузка врачей, финансовые показатели, статистика по пациентам, эффективность отделений. |

### Управление

| Модуль | Описание |
|--------|----------|
| **Расписание** | Управление приёмами, сменами врачей, загрузкой кабинетов. Запись пациентов с учётом доступности. |
| **Биллинг** | Выставление счетов, приём платежей, финансовая аналитика. Учёт по отделениям и врачам. |
| **Персонал** | Профили сотрудников, управление ролями и правами, учёт посещаемости, расписание смен. |
| **Палаты** | Управление койками: размещение пациентов, статус занятости, история назначений. |
| **Аудит** | Полный журнал действий всех пользователей. Кто, когда, что изменил — для соответствия медицинским стандартам. |

### Технологии

| Модуль | Описание |
|--------|----------|
| **Телемедицина** | Видеоконсультации врач-пациент через Daily.co. Текстовый чат, история сообщений. Без установки приложений. |
| **AI ассистент** | Голосовое управление порталом пациента. Speech-to-Text (Whisper), понимание команд (Gemini/DeepSeek), выполнение действий голосом. |
| **IoT мониторинг** | Мониторинг жизненных показателей в реальном времени: пульс, SpO2, температура, давление, детектор падений. Мгновенные алерты медсёстрам через WebSocket. 10 типов датчиков, кнопка вызова. |
| **BMS (управление зданием)** | IoT-управление инфраструктурой: климат (кондиционеры, вентиляция), освещение, замки дверей, пожарная безопасность, лифты. Автоматизация по расписанию и условиям. |
| **Портал пациента** | Личный кабинет пациента: запись на приём, результаты анализов, видеоконсультации, упражнения для реабилитации, оплата счетов, голосовой ассистент. Вход по телефону. |

## Ключевые возможности

### Для главврача
- Полная аналитика по клинике в реальном времени
- Контроль загрузки врачей и палат
- Аудит всех действий персонала
- Финансовые отчёты
- Управление зданием и оборудованием

### Для врача
- Быстрые назначения и рецепты
- AI-помощник для диагностики
- Видеоконсультации с пациентами
- Полная история пациента в один клик
- Планы лечения с контролем выполнения

### Для медсестры
- Мониторинг витальных показателей в реальном времени
- Автоматические алерты при критических значениях
- Электронные листы назначений
- Мгновенный вызов через кнопку

### Для регистратуры
- Регистрация пациента за 2 минуты (OCR + распознавание лица)
- Управление расписанием всех врачей
- Распределение по палатам
- Приём платежей

### Для пациента
- Личный кабинет — запись на приём, результаты анализов
- Видеоконсультация с врачом из дома
- Упражнения для реабилитации с AI-трекером
- Голосовой ассистент — навигация голосом
- Оплата счетов онлайн

## Технологический стек

### Backend
- **Python 3.11** + **FastAPI** — высокопроизводительный API
- **PostgreSQL 16** — надёжная база данных
- **Redis 7** — кэширование и сессии
- **SQLAlchemy 2.0** (async) — ORM
- **Celery** — фоновые задачи (email, уведомления)
- **MinIO** — S3-совместимое хранилище файлов
- **Alembic** — миграции базы данных

### Frontend
- **React 18** + **TypeScript** — типизированный UI
- **TanStack Router** — file-based маршрутизация
- **TanStack Query** — серверный стейт
- **Zustand** — клиентский стейт
- **Tailwind CSS** — утилитарные стили
- **Framer Motion** — анимации
- **Recharts** — графики и диаграммы

### AI/ML
- **Google Gemini** — NLU для голосового ассистента
- **Whisper** — Speech-to-Text
- **MediaPipe** — определение позы для упражнений

### DevOps
- **Docker** + **Docker Compose** — контейнеризация (8 сервисов)
- **Nginx** — reverse proxy
- **Railway** — облачный деплой

## Быстрый старт

```bash
# Клонировать репозиторий
git clone <repo-url>
cd telemed-v2-super

# Скопировать конфигурацию
cp .env.example .env

# Запустить все сервисы
docker-compose up -d

# Система автоматически:
# 1. Запустит PostgreSQL, Redis, MinIO
# 2. Применит миграции БД
# 3. Заполнит начальные данные (8 seed-скриптов)
# 4. Запустит backend и frontend
```

Открыть:
- **Админ-панель:** http://localhost (логин для персонала)
- **Портал пациента:** http://localhost/portal/login
